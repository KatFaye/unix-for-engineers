TLDR - sudo
==========

Overview
--------

[sudo] - run command with elevated privileges 

	$ sudo apt-get install program

Examples
--------

**-b** - run given command in background
	$ sudo -b apt-get install program

**-V** - update user's cached credentials; does not run a command

	$ sudo -V
