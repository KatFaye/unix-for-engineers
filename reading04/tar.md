TLDR - tar
==========

Overview
--------

[tar] - archiving utility

	$ tar czvf file.tar.gz directory1

Examples
--------

**czvf** - creates archive from directory with verbose output
	$ tar czvf file.tar.gz directory1

**-x** - decompress

	$ tar xzvf compressedFile
