TLDR - ps
==========

Overview
--------

[ps] displays current processes

	$ ps

Examples
--------

**-a** - show processes of all users

	$ ps -a

**-W** - shows both Windows and cygwin processes

	$ ps -W

**-l** - shows more information

	$ps -l
