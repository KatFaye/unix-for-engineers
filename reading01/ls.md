TLDR - ls
==========

Overview
--------

[ls] prints directory content (lists files in a folder)

Examples
--------

**-a** - display normally-hidden files

	$ ls -a reading01

**-lat** - displays in order of last changes

	$ ls -lat 

**-S** - displays in order of size (biggest first)

	$ ls -S
