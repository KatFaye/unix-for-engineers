TLDR - gcc
==========

Overview
--------

[gcc] - used to compile C programs

	$ gcc hello.c

Examples
--------

**-o** - Allows you to set a custom name for the executable
	$ gcc -o hello hello.c

**-c** - Separate compiling and linking (useful when using several files for one executable)

	$ gcc -c hello.c
