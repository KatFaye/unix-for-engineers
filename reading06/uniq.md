TLDR - uniq
==========

Overview
--------

[uniq] - report/omit repeated lines

	$ uniq [option] [file]

Examples
--------

**-d** - only shows duplicate lines

	$ uniq -d myFile

**-u** - only shows duplicate lines

	$ uniq -u myFile
