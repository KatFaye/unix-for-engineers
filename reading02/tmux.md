TLDR - tmux
==========

Overview
--------

[tmux] - allows use of persistent working states on remote servers

	$ tmux

Examples
--------

**new -s sessionName** - creates a new tmux session

	$ tmux new -s mySession

**a** - connects to first available session

	$ tmux a -t
