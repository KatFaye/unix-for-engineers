TLDR - ping
==========

Overview
--------

[ping] - check if able to connect to a given network host

	$ ping 149.165.180.102

Examples
--------

**-s packetsize** - sets how large packet sent is

	$ ping -s 128 ipaddr

**-c count** - stops pinging after sending specified amount of packets

	$ ping -c 4 ipaddr
